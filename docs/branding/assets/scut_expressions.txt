SCUT — The Crab's Friend | ASCII Art Expression Sheet
CRAB Brand System | Terminal Core Aesthetic
Scut is a tiny crablet companion to the Canonical CRAB.
He rides on the big crab's shell, scuttles alongside, or appears
alone in status bars, receipts, and UI nudges.

Colors: Cyan (81) body, White (15) eyes/mouth, Amber (208) when excited.
Render with: cat scut_expressions.txt

NOTES
================================================================================
- All expressions fit within an 8-line by 20-column bounding box
- Eyes toggle between: o o (normal), - - (sleepy), O O (alert), . o (thinking)
- Mouth toggles between: > (normal), ^ (happy), o (shock), - (sleepy), ⏝ (calm)
- Claws toggle between: | | | (neutral), \ | / (celebrating), > | < (working)
- Add "\033[38;5;208m" before Scut for amber coloring
- Add "\033[38;5;81m" before Scut for cyan coloring
- Use "\033[0m" after to reset color
