CRAB — Canonical ASCII/ANSI Art
=================================
Terminal Core Aesthetic | Neo-ANSI Revival | v1.1
Render with: cat crab_canonical.txt

All colors use ANSI 256-color escape sequences.
A = \033[38;5;208m (AMBER)  |  C = \033[38;5;81m (CYAN)
W = \033[38;5;15m  (WHITE)  |  D = \033[38;5;245m (DIM)
K = \033[40m         |  0 = \033[0m  (RESET)

Small  (8 lines)  → Favicon, inline icon, receipt seal
Medium (14 lines) → Panel header, dashboard chrome, bus card
Large  (19 lines) → Splash screen, release notes hero, ceremonial

================================================================================
ANATOMY GUIDE — Why This Reads as a Crab
================================================================================

CLAWS      \\     /     ← The dominant feature. Raised, ready.
            )___(       ← Pincer joint. One claw may be larger.

EYES     .--(o o)--.   ← On stalks, not embedded in body. Vital for
                          "watching" — perfect for a governance protocol.

CARAPACE /  .-----.  \ ← Wide, shield-shaped. Ridge down center.
         /  / ~   ~ \  \ ← Wave texture = shell segmentation.

MOUTH    (  |    >    |  ) ← Ventral, small. Not a smile — a task focus.

LEGS        /| | |\      ← Splayed outward, not vertical. 5 pairs =
           / | | | \       decapod (10 legs), biologically accurate.
          (  | | |  )
           \ | | | /
            \|_|_|_/

SCUT       ___          ← A miniature version of the same anatomy,
          /o o\           scaled down. Same eye/mouth/leg grammar.
          \___/
          | | |

================================================================================
COMPOSITION RULES
================================================================================
1. Symmetry = Neutral/calm     (legs | | |)
2. Asymmetry = Active/emotion  (one claw larger, legs > < or \ /)
3. Eye size:  o → O → -        (normal → alarmed → closed)
4. Mouth:     > → ^ → o → -    (neutral → happy → surprised → relaxed)
5. Scut mirrors the big crab's expression at 1:2 scale
6. Amber (208) = shell, authority     Cyan (81) = friend, companion
7. White (15) = eyes, focal points   Dim (245) = legs, ground
