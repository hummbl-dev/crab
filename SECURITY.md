# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this HUMMBL repository, please report it responsibly.

1. Email: reuben@hummbl.io
2. Do not open a public GitHub issue for security vulnerabilities.
3. Include a description, steps to reproduce, and potential impact.
4. Expect an initial response within 48 hours.

## Scope

This policy covers this repository and its maintained default branch.
